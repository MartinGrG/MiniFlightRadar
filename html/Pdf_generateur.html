PDF_generateur module¶

Ce script décrit l’objet PDF et donne les fonctions associées

class FlightRadar.PDFgenerator.Pdf_generateur.Pdf(map_chemin, classe)¶

Bases: FPDF

Cette classe étend les fonctionnalités de FPDF pour générer un document PDF personnalisé.

Parameters:

  • titre (Str) – Titre du document PDF, par défaut “Titre par défaut”.

  • infos_vol (list) – Informations sur le vol sous forme de liste, contenant différents détails comme l’index, le callsign, les dates de départ et d’arrivée, la compagnie, le modèle d’avion, etc.

  • infos_emission (list) – Informations sur les émissions de carbone, comprenant des données sur les moteurs, les émissions associées à chaque moteur, les modèles d’avions, etc.

  • graphique_emission (Str) – Chemin vers le graphique des émissions de carbone.

  • map_chemin (Str) – Chemin vers l’image de la carte capturée.

  • classe (Str) – Classe de siège pour laquelle les émissions sont calculées, par exemple économique, affaires, etc.
